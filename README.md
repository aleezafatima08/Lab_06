# Fitts's Law - Reaction Time Experiment

## Overview
An interactive Human-Computer Interaction (HCI) reaction test game built using HTML, CSS, and JavaScript. It evaluates how quickly and accurately users respond to visual stimuli in a dynamic interface.

This project is developed as part of an HCI lab assignment to study human reaction time, visual attention, and motor response behavior.

## Objective
The objective of this game is to identify and click the correct colored circles that match the displayed target color as quickly as possible while avoiding incorrect selections.

## HCI Focus
This game demonstrates key Human-Computer Interaction principles:

- Reaction time and motor response  
- Visual search and selective attention  
- Fitts’ Law (target selection efficiency)  
- Hick’s Law (decision-making complexity)  

## Outcome
The system records and displays user performance including:

- Score (number of correct selections)  
- Miss count (number of wrong clicks)  
- Reaction time (measured in milliseconds)
